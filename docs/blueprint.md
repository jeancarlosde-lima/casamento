# **App Name**: Eloisa & Jean - Wedding Invitation

## Core Features:

- Event Details: Display key wedding details such as date, time, and venue.
- Interactive Map: Include a map or direct link to the venue for easy navigation.
- RSVP Form: Allow guests to RSVP online.
- Our Story: Present the love story of Eloisa & Jean.
- Gift Registry: Link to the couple's gift registry.
- Travel Assistant: AI powered tool to suggest travel and lodging options near the venue, providing different possibilities

## Style Guidelines:

- Primary color: Soft Lavender (#D0B8E5) to represent romance and elegance.
- Background color: Light Beige (#F5F5DC) for a warm, inviting feel.
- Accent color: Muted Rose (#BC8F8F) to highlight important elements.
- Headline font: 'Playfair', serif, for an elegant, fashionable feel. Body font: 'PT Sans', sans-serif.
- Use delicate, minimalist icons related to weddings (e.g., rings, hearts, flowers).
- Employ a clean and modern layout with plenty of white space to avoid clutter.
- Use subtle animations like a gentle fade-in effect when the page loads.
